This dashboard allows you to manually trigger the synchronization between Asana tasks with the schedule tag and Google Calendar.

How it works: When you click the sync button, the system will look for Asana tasks that have both:
  • A due date assigned
  • The schedule tag
For each matching task, it will create a calendar event at the due date time.
